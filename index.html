1 - Integração Javascript/HTML

O Javascript consegue acessar todos os elementos da página HTML, usando os mesmo tipos de seletores que o CSS. No Javascript, usaremos os elementos HTML em 2 tipos de tarefas, leitura/escrita e interação. Existem diversas formas para acessar os elementos HTML, mas neste exercício usaremos as seguintes funções presentes no Javascript:

  • document.getElementById("id-do-elemento")
    • Seleciona um elemento através do seu ID (semelhante ao #id-do-elemento no CSS)
  • document.getElementsByClassName("classe-do-elemento")
    • Seleciona uma lista de elementos através da sua classe (semelhante ao .classe-do-elemento no CSS)
  • document.getElementsByTagName("tag-do-elemento")
    • Seleciona uma lista de elementos através do seu tipo de tag (semelhante ao tag-do-elemento no CSS)

2 - Elemento HTML no Javascript

Quando recuperamos um elemento usando uma função como o document.getElementById, obtemos um HTMLElement no Javascript. Podemos guardar esse elemento em uma variável e reutilizá-lo diversas vezes.

let elemento = document.getElementById("id-do-elemento")

Este elemento poderá ser utilizado para ler e alterar diversas informações. As principais formas de acesso das informações, estão classificadas em 3 categorias:

  • elemento.innerHTML
    • Permite consultar e alterar o conteúdo html do elemento.
  • elemento.atributo
    • Permite consultar e alterar o conteúdo de qualquer atributo presente no html (como class, value e href).
  • elemento.style.propriedade
    • Permite consultar e alterar o conteúdo de qualquer propriedade do CSS aplicado àquele elemento.

3 - Utilizando o valor de retorno do elemento.value

O retorno do elemento.value será textual, portanto, uma string. O operador de soma (+) irá fazer a concatenação de textos ao final da string.

let elemento = document.getElementById('id-do-elemento')
let texto = elemento.value
console.log(texto) // imprime "texto do input"
texto = texto + " " + "atualizado"
console.log(texto) // imprime "texto do input atualizado"

Podemos utilizar o elemento.value como um número, mas para isso vamos precisar dizer para o Javascript que ele deve interpretar aquele texto. A função parseInt(texto) irá nos ajudar com isso.

let elemento = document.getElementById('id-do-elemento')
let texto = elemento.value
let numero = parseInt(texto)
console.log(numero) // imprime "99"
numero = numero + 10
console.log(numero) // imprime "109"

// Se tentarmos utilizar texto (sem fazer o parseInt)
console.log(texto) // imprime "99"
texto = texto + 10
console.log(texto) // imprime "9910" (concatenou "99" com "10")

4 - Eventos

Outra forma de integração de código Javascript com o HTML é através de eventos. Eventos são ações produzidas por usuários na navegação da página que podem ser utilizadas pelo Javascript para disparar uma função.

A forma mais comum de interação é através do clique de um botão. Podemos fazer o clique de um botão acionar uma função do Javascript usando o código:

let botao = document.getElementById('id-do-botao')
botao.onclick = function() {
alert("botão clicado")
}

Existem outros tipos de eventos que podem ser capturados, como o foco em um elemento input, a ação de passar o mouse sobre um elemento, apertar uma tecla no teclado, entre outros. Mas, vamos utilizar ao longo deste exercício apenas o onclick.
